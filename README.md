# 💼 Yashi Gupta – Portfolio Website

A fully responsive and interactive **portfolio website** showcasing my journey as a **Computer Engineering student** and creative **front-end developer**. Designed with clean aesthetics, smooth transitions, and a focus on user experience, it highlights my projects, skills, and personal brand in one place.

✨ Introduction
Welcome to my portfolio website – a space where technology meets creativity.

I’m Yashi Gupta, a dedicated and curious Computer Engineering student with a deep passion for front-end development, UI design, and interactive web experiences. This portfolio is more than just a collection of my work – it's a personal canvas that reflects my growth, vision, and the projects I’ve brought to life through code and creativity.

Crafted using HTML, CSS, and JavaScript, this website demonstrates my ability to design elegant, responsive interfaces that not only look good but also offer intuitive user interaction. The entire structure is hand-coded, showcasing my attention to detail, problem-solving mindset, and understanding of clean and maintainable code practices.

The portfolio highlights a variety of projects that represent both technical challenges and creative exploration. From pastel-themed landing pages like FrostyDelight, to a fully functional academic Student Planner, and the genre-rich The Novel’s Nest for book lovers — each project is a result of thoughtful design, precise implementation, and a continuous drive to improve.

I believe in the power of simple yet impactful design. Every section in this portfolio is structured to give visitors a smooth, engaging experience across all screen sizes. Beyond the visual, it reflects my personality as a builder — someone who finds joy in transforming ideas into tangible, interactive results.

This portfolio also serves as a foundation for future enhancements. I’m currently working on adding more interactivity, such as contact forms, downloadable resume integration, and refined animations. It's a living project that evolves as I learn and grow.

Whether you’re here to review my work, collaborate, or just explore, I’m glad you stopped by. Let’s build something great together!
